redis Set(集合)


redis 的 Set 是 String 类型的无序集合，元素不允许重复。

redis 中集合是通过哈希表实现的，所以添加，删除，查找的复杂度都是 O(1)。

集合中最大的元素数为 232 - 1 (40多亿)。
