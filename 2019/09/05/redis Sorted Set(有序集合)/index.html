redis Sorted Set(有序集合)


redis Sorted Set 有序集合是 string 类型元素的集合，元素不允许重复。


有序集合中的每个元素都会关联一个数值型的分数。redis 正是通过分数来为集合中的成员进行从小到大的排序。


有序集合的成员是唯一的（不可重复）,但分数(score)可以重复。



sorted set 也提供了计算交集、并集的命令，此处不再一一介绍。
